how to setup this build page? only once for initial setup

completely exit from source repository!
in outside of the source repo,

mkdir build (whatever name to build the site)
cd build
git init
git clone git@github.com:saugkim/sphinx.git html
go to html
git branch gh-pages

git symbolic-ref HEAD  refs/heads/gh-pages
rm .git/index
git clean -fdx

git branch (confirm branch on gh-pages)

--> Edit Makefile in source directory and change build location 
default is BUILDDIR = build
to 	   BUILDDIR = ../../build
 
--> build in local repository (by cmake or manually sphinx-build)
mingw32-make (in Anaconda console in my case)

git add .
git commit -m "first built"
